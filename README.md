# 🍀 DESAFIO-IN8-DEV

### 📂 Sobre 
 Esse projeto é responsavel por cadastrar e lista os dados sobre os estagiários. Projeto simples, integrando o client com o server. 
 
### 🛠 Construido 
- **Server**
  - Node.js
  - KNEX
  - Cors
  - Helmet
  - MySql
  - Express
- **Client**
  - React.js
  - HTML
  - CSS
  - REACT-ROUTER-DOM
  - YUP
  - REACT-HOOK-FORMS

### 🎑 Clonar o projeto

Node.js LS
React.Js LS

```bash 
git clone : https://github.com/kaiquye/DESAFIO-IN8-DEV.git
npm install
npm start
```
