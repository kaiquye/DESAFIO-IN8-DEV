# 🍀 DESAFIO-IN8-DEV

### 📂 Sobre 
 Esse projeto é responsavel por cadastrar e lista os dados sobre os estagiarios. Projeto simples, que o client se integra com o server.
 
### 🛠 Construido 
- **Server**
  - Node.js
  - KNEX
  - Cors
  - Helmet
  - MySql
  - Express
- **Client**
  - React.js
  - HTML
  - CSS
  - REACT-ROUTER-DOM
  - YUP
  - REACT-HOOK-FORMS

### 🎑 Clona o projeto
```bash 
git clone : https://github.com/kaiquye/PROJETO-SEMINOVOS.git
npm install
npm start
```
